## 项目启动

### 方式一：VS Code Live Server（推荐）

1. 用 VS Code 打开项目文件夹
2. 安装 Live Server 插件
3. 右键点击 `index.html` → "Open with Live Server"
4. 浏览器自动打开 `http://127.0.0.1:5500`

### 方式二：Python 简易服务器

```bash
# Python 3
python -m http.server 8080

# 访问 http://localhost:8080
```

### 方式三：Node.js 服务器

```bash
# 安装 serve
npm install -g serve

# 启动
serve .

# 访问 http://localhost:3000
```

#### 命名规范


## 组件开发

### 创建新组件

#### 1. 基础组件模板

```javascript
// ===== 新组件 =====
const MyComponent = {
  name: 'MyComponent',
  
  // 接收的属性
  props: ['propName'],
  
  // 发出的事件
  emits: ['eventName'],
  
  // 组件数据
  data() {
    return {
      localData: null,
    };
  },
  
  // 计算属性
  computed: {
    computedValue() {
      return this.localData + 1;
    },
  },
  
  // 方法
  methods: {
    handleClick() {
      this.$emit('eventName', this.localData);
    },
  },
  
  // 模板
  template: `
    <div class="my-component">
      <button @click="handleClick">点击</button>
    </div>
  `,
};
```

#### 2. 注册组件

```javascript
// 在 app.js 中注册
const App = {
  components: {
    MyComponent,
    'my-component': MyComponent,
  },
  // ...
};
```

### 组件通信

#### 父传子（Props）

```javascript
// 父组件
<child-component :data="parentData"></child-component>

// 子组件
props: ['data']
```

#### 子传父（Events）

```javascript
// 子组件
this.$emit('update', newValue);

// 父组件
<child-component @update="handleUpdate"></child-component>
```

#### 全局状态（Store）

```javascript
// 读取状态
const students = Store.state.students;

// 调用方法
Store.addPoints(1, 50, '奖励');
```

## 常见问题

### Q1: 修改代码后没有生效？

**原因**：浏览器缓存了旧代码

**解决**：
- 按 `Ctrl+F5` 强制刷新
- 或打开 DevTools → Network → Disable cache

### Q2: localStorage 数据混乱？

**解决**：清除数据重新开始

```javascript
// 控制台执行
localStorage.removeItem('pet_system_students');
localStorage.removeItem('pet_system_tasks');
location.reload();
```

### Q3: 如何备份数据？

```javascript
// 导出数据
const data = {
  students: JSON.parse(localStorage.getItem('pet_system_students')),
  tasks: JSON.parse(localStorage.getItem('pet_system_tasks'))
};
console.log(JSON.stringify(data));
// 复制输出的 JSON 保存到文件

// 导入数据
const backup = {/* 之前保存的 JSON */};
localStorage.setItem('pet_system_students', JSON.stringify(backup.students));
localStorage.setItem('pet_system_tasks', JSON.stringify(backup.tasks));
location.reload();
```

### Q4: 如何添加更多初始数据？

编辑 `js/store.js` 中的 `init()` 方法：

```javascript
init() {
  // ... 现有代码
  
  // 添加更多初始学生
  const moreStudents = [
    { id: 7, name: '小新', username: 'xiaoxin', ... },
  ];
  this.state.students.push(...moreStudents);
  
  // ...
}
```

## 性能优化

### 当前优化

1. **Vue 响应式**：只追踪必要的数据变化
2. **CSS 动画**：使用 transform 和 opacity，利用 GPU 加速
3. **事件委托**：减少事件监听器数量

### 可优化方向

1. **虚拟列表**：学生/任务过多时使用
2. **防抖节流**：频繁触发的事件（如搜索）
3. **懒加载**：图片资源（如果使用）
